eMart IT Chatbot — Embed Test Page

This page demonstrates the embed script. The chat bubble below is loaded via the embed script — exactly how it will work on any client website.

Embed code for this demo:

<script src="https://emartit.github.io/eMart-IT-Chatbot/widget.js?client=demo"></script>

To add this chatbot to any website, paste the line above before the closing </body> tag.
